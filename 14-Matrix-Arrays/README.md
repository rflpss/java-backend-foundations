# 🔢 Two-Dimensional Arrays (Matrices)

A practical grid manipulation algorithm to locate neighbors in a two-dimensional space.

## 🎯 Objective
To read matrix dimensions ($M \times N$) and its elements, locate all occurrences of a specific integer, and print its surrounding neighbors (left, right, top, and bottom).

## 🧠 Concepts Applied
* **Nested Loops:** Mastering row and column traversal using nested `for` statements.
* **Boundary Validation:** Implementing index safety checks to avoid `ArrayIndexOutOfBoundsException` when querying edge elements.
