# ➕ Vector Sum and Average

A straightforward Java application that demonstrates array population, mathematical accumulation, and formatted data output.

## 🎯 Objective
To prompt the user for a specific number of floating-point values, store them inside an array, and then process the collection to print all the elements inline, followed by their total sum and arithmetic average.

## 🧠 Key Concepts Applied
* **Mathematical Accumulation:** Using a `for` loop to traverse the array and keep a running total of the elements (`sum = sum + vect[i]`).
* **Data Aggregation:** Calculating the average based on the accumulated sum and the array's length.
* **Formatted Output (`printf`):** Manipulating console output to display array elements sequentially on a single line with specific decimal precision (`%.1f `), ensuring a clean user interface.
