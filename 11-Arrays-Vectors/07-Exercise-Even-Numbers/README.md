# 🔢 Even Numbers Filter

A Java console application focused on conditional array iteration and data filtering using mathematical operators.

## 🎯 Objective
To read an 'N' amount of integers into an array, then iterate through it to print all the even numbers sequentially. Finally, the program must output the total count of even numbers found.

## 🧠 Key Concepts Applied
* **Modulo Operator (`%`):** Utilizing `% 2 == 0` to accurately determine if an array element is an even number.
* **Inline Printing:** Formatting the output to display all filtered values on a single line separated by spaces.
* **Accumulator Variables:** Managing a separate integer counter (`evenNumber`) that increments only when the specific condition is met during the iteration.
