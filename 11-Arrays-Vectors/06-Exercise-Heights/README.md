# 📊 Demographics Analysis - Parallel Arrays

A Java application that collects demographic data (name, age, and height) and performs statistical analysis using parallel arrays.

## 🎯 Objective
To process data from 'N' individuals, storing each attribute in its respective array. The program then calculates the average height of the group, determines the percentage of people under 16 years old, and lists their names.

## 🧠 Key Concepts Applied
* **Parallel Arrays:** Using independent arrays (`String[]`, `int[]`, `double[]`) mapped together by a common index variable (`i`) to represent complete data records.
* **Type Casting:** Forcing integer variables into double format during arithmetic operations to ensure accurate percentage calculations and avoid integer division truncation.
* **Complex Iteration:** Utilizing multiple loop structures to separate data entry, accumulation, conditional counting, and specific data retrieval.
