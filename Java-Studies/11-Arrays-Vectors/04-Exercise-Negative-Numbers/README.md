# ➖ Negative Numbers Filter - Array Practice

A Java console program designed to practice basic array manipulation, data population, and sequential iteration with conditional filtering.

## 🎯 Objective
To read a maximum of 10 integer numbers provided by the user, store them in an array, and then process the array to display only the negative numbers. 

## 🧠 Key Concepts Applied
* **Dynamic Array Sizing:** Instantiating an array with a size (`n`) defined by the user at runtime.
* **Separation of Concerns:** Using distinct `for` loops to isolate the data input process from the data processing/filtering logic.
* **Conditional Iteration:** Traversing the array structure and applying an `if (vect[i] < 0)` validation rule to manipulate specific elements based on business requirements.
