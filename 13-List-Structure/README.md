# 📜 Dynamic Employee List Management

An application focused on dynamic data structures using Java `List` and Wrapper classes.

## 🎯 Objective
To register employees with unique IDs, names, and salaries, allowing a dynamic salary increase based on a percentage input.

## 🧠 Concepts Applied
* **Java List Interface (`ArrayList`):** Implementing a dynamic array that automatically resizes, unlike primitive arrays.
* **Lambda & Streams:** Utilizing `.stream().filter()` and `.findFirst()` to look up objects inside a collection using predicates efficiently.
* **Encapsulation:** Protecting data integrity via class attributes, constructors, and proper getters/setters.
