# 🧬 Inheritance & Polymorphism

Repository dedicated to the study and practical exercises of the Inheritance and Polymorphism module in Java.

## 🎯 Objective
To master the core pillars of code reusability and dynamic behavior in Object-Oriented Programming (OOP) using Java.

## 📂 Projects in this Module

### 1. [Product Tags System](./inheritance-products)
A system that reads data for common, used, or imported products, calculates dynamic costs (like customs fees), and generates custom price tags.
* **Key Concepts:** Class Inheritance, Polymorphism, Method Overriding (`@Override`), and Polymorphic Collections (`List<Product>`).

### 2. [Tax Calculator System](./tax-calculating)
A program that calculates the tax paid by different types of taxpayers (Individuals and Companies) based on distinct business rules.
* **Key Concepts:** Abstract Classes, Abstract Methods, Inheritance, and processing data dynamically through a polymorphic list (`List<TaxPayer>`).

---

## 🧠 Core OOP Concepts Covered
* **Inheritance:** Allows a class (subclass) to inherit attributes and methods from another (superclass), promoting code reuse.
* **Polymorphism:** The ability of an object to take on many forms, allowing the executed method to change dynamically at runtime based on the object type.
* **Abstract Classes & Methods:** Utilizing base classes that cannot be instantiated directly, enforcing specific implementation rules in subclasses without defining a default behavior.
