# 🧬 Inheritance & Polymorphism

Repository dedicated to the study and practical exercises of the Inheritance and Polymorphism module in Java.

## 🎯 Objective
To master the core pillars of code reusability and dynamic behavior in Object-Oriented Programming (OOP).

## 🧠 Core Concepts
* **Inheritance:** Allows a class (subclass) to inherit attributes and methods from another (superclass), promoting code reuse.
* **Polymorphism:** The ability of an object to take on many forms, allowing the executed method to change dynamically at runtime based on the object type.
* **Method Overriding (`@Override`):** Redefining a base class method to customize the behavior in child classes.
